# AI-ML Internship Projects

This repository contains Machine Learning projects completed during my AI-ML Internship.

## Projects

### 1. Heart Disease Prediction
A machine learning model that predicts whether a patient is likely to have heart disease based on medical data.

### 2. House Price Prediction
This project predicts house prices based on features like area, number of rooms, and location using regression techniques.

### 3. Future Stock Price Prediction
A model that analyzes historical stock data to predict future stock prices.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
